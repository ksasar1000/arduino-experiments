# Clean GitHub

uBlock Origin filters that remove promotional content from GitHub READMEs

## One-Click Install

[![Add to uBlock](https://img.shields.io/badge/Click_to_Install-uBlock_Origin-800020?style=for-the-badge)](https://subscribe.adblockplus.org/?location=https://raw.githubusercontent.com/clean-github/filters/main/list.txt&title=GitHub%20Cleanup)

## What Gets Blocked

- "Sponsor this project" buttons
- Newsletter signup forms  
- "Try our premium service" boxes
- Affiliate marketing links
- Other README clutter

The filters are updated regularly as new promotional patterns emerge.

## Help Improve

Found something that should be blocked? Submit a PR with the new filter rule.
